SPF check : Vérifier l'enregistrement SPF

Avec le vérificateur SPF d'EXPERTE.com, vous pouvez déterminer rapidement et facilement les enregistrements SPF d'un domaine. Il suffit pour cela de saisir le domaine souhaité dans le champ ci-dessus.

En réponse, vous obtenez un tableau contenant les différents éléments de l'enregistrement SPF.

Enregistrement SPF - Structure

Avec un enregistrement SPF, vous pouvez définir quelles adresses IP sont autorisées à envoyer un e-mail sous votre nom (domaine).

Un enregistrement SPF commence toujours par le numéro de version actuel (« v=spf1 »), qui peut être suivi de différentes expressions. Une expression se présente sous la forme « Qualificateur » + « Mécanisme ».

Les qualificateurs sont les suivants :

Q.RésultatDescription

+

Pass

expéditeurs autorisés

-

Fail

expéditeurs non autorisés

~

SoftFail

expéditeur non autorisé ; le destinataire doit quand même recevoir le message

?

Neutre

L'expéditeur doit être classé comme neutre

Les principaux mécanismes sont les suivants :

M.Description

a

Correspond si le domaine a un enregistrement A correspondant à l'expéditeur

ip4

Correspond si l'IP de l'expéditeur se trouve dans cette plage IPv4

ip6

Correspond si l'IP de l'expéditeur se trouve dans cette plage IPv6

mx

Correspond si le domaine a un enregistrement MX qui correspond à l'expéditeur

include

Fait référence à la politique d'un autre domaine. Si celle-ci est réussie, le mécanisme est réussi

all

Correspond toujours

Enregistrement SPF - Exemples

Nous souhaitons donner ci-après quelques exemples de politiques :

ExempleDescription

+ip4:35.190.247.0/24

Accepter tous les mails de la plage IP 35.190.247.0/24

include:_spf.google.com

Utiliser la politique SPF de google.com pour ce domaine

~all

Marquer tous les autres expéditeurs comme SoftFails
