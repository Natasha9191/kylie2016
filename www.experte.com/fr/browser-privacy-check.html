Vérification de la confidentialité du navigateur : Quelles sont mes traces sur internet ?

Avec le Contrôle de la Confidentialité du Navigateur d'EXPERTE.com, vous pouvez découvrir les traces que vous ou votre navigateur laissez derrière vous lorsque vous surfez sur internet. Le test a pour but de faire prendre conscience des données qui peuvent être utilisées par les sites internet et les annonceurs pour créer un profil sur vous ou pour suivre vos activités sur plusieurs sites.

Pour commencer le test, veuillez cliquer sur le bouton ci-dessus.

Le contrôle du navigateur collecte 7 types de données :

  • Connexion
    Le contrôle du navigateur tente de déterminer votre adresse IP locale (via WebRTC) et votre serveur DNS en plus de votre adresse IP publique. Nous vérifions également si vous accédez au site via un proxy connu ou Tor et si votre adresse IP a attiré une attention négative dans le passé.

  • Localisation
    Nous utilisons votre adresse IP pour déterminer votre localisation et d'autres données sur votre emplacement.

  • Annonces et traqueurs
    Nous vérifions si votre navigateur autorise les publicités et les traceurs.

  • Médias sociaux
    Nous utilisons une astuce pour identifier les réseaux sociaux auxquels vous êtes actuellement connecté.

  • Matériel
    Nous recueillons des informations sur le matériel que vous utilisez, notamment le processeur et la carte graphique, ainsi que l'état actuel de la batterie.

  • Logiciels
    En plus du matériel, nous déterminons également les logiciels utilisés, notamment le système d'exploitation, le navigateur et les plugins utilisés.

  • Empreinte digitale
    À partir de toutes ces informations, nous générons une empreinte digitale spécifique à votre navigateur. Grâce à un test, nous vous montrons que vous pouvez également être suivi de façon anonyme.

Le Contrôle de Confidentialité donne une explication pour chaque catégorie, et propose des solutions afin d'empêcher le suivi.

Protection des données

Nous prenons la protection de vos données très au sérieux. Cependant, afin de déterminer les informations de confidentialité, nous devons faire des demandes à certains services externes tels que Google Analytics. Veuillez ne pas commencer le test si vous y êtes opposé. À l'exception de l'empreinte digitale du navigateur (et ce uniquement après confirmation active de votre part), nous ne stockons aucune des données déterminées.
